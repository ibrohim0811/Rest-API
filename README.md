# 🚀 Django REST API Project

Ushbu loyiha **Django REST Framework (DRF)** yordamida yaratilgan backend API hisoblanadi. Loyiha doirasida foydalanuvchilarni autentifikatsiya qilish, ma'lumotlar bilan ishlash va API integratsiyalari amalga oshirilgan.

## ✨ Xususiyatlari
* **RESTful API** - Standart HTTP metodlari (GET, POST, PUT, DELETE).
* **Authentication** - JWT yoki Token asosidagi autentifikatsiya.
* **Face Auth** (Agar qo'shgan bo'lsang) - Yuzni aniqlash orqali kirish tizimi.
* **Postman Ready** - API so'rovlarini tekshirish uchun tayyor endpointlar.

---

## 🛠 Texnologiyalar
* **Backend:** Python 3.x, Django 5.x
* **API:** Django REST Framework
* **Database:** PostgreSQL (yoki SQLite)
* **Tools:** Postman, AnyDesk (debugging uchun)
